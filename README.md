# 天象分析 · 风光摄影天气预报

一个为风光摄影师打造的天象拍摄条件预报工具，预测最佳出片时机。

## 功能

- 🔥 **火烧云**：朝霞 / 晚霞拍摄条件预报
- ☁️ **云海**：山顶云海概率分析
- 🏔️ **日照金山**：点选山峰即可分析金山照亮时机
- ⭐ **星空 / 银河**：含月相、光污染、银河核心升落与最佳窗口
- 📅 **机会日历**：未来 7 天最佳摄影日排行
- 🏠 **首页驾驶舱**：今晚明早一眼速览
- 📡 **多模型对比**：GFS / ECMWF / ICON 三模型一致性
- 📐 **真实地平线**：估算山体遮挡对日出日落时刻的影响

## 数据来源

- 天气数据：Open-Meteo API（GFS / ECMWF / ICON）
- 地图：OpenStreetMap
- 海拔 / 地形：Open-Meteo Elevation + OpenStreetMap

## 使用方式

直接打开 `index.html` 即可，无需安装。所有数据本地存储，不上传任何信息。
